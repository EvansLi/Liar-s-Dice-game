# 吹牛遊戲 & 翻譯機器人

## 簡介
### 【吹牛機器人】
* 利用群組功能玩吹牛遊戲。
* 使用者透過群組加入並開始遊戲。
* 透過私人視窗獲取點數，並進行競猜。

### 【翻譯機器人】
* 透過直接上傳圖片，即可將圖中文字辨識並翻譯成中文。
* 透過「泰語 你好」翻譯出泰語的你好。

## 相關工具
* Python Flask
* Line Bot 
* Heroku
* 百度翻譯API
* Google Drive API 圖片辨識
